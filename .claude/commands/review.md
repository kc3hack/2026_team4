人が必要と判断したときのみ使用するレビューコマンドです。

以下の観点に **限定** してレビューしてください。それ以外の指摘は不要です。

## レビュー観点

1. **MVVM の責務違反がないか**
   - View にロジックが入っていないか
   - ViewModel に View 依存のコードが入っていないか
   - Model に状態管理が入っていないか

2. **View が重くなっていないか**
   - 1つの View に責務を詰め込みすぎていないか
   - 不要なネストが深くなっていないか

3. **ハッカソン向けにやりすぎていないか**
   - 不要な抽象化・レイヤー分割をしていないか
   - Repository / UseCase など禁止パターンを導入していないか
   - 過剰な設計になっていないか

## 出力形式

- 問題がなければ「問題なし」とだけ回答する
- 問題がある場合はファイル名と該当箇所を明示して簡潔に指摘する
